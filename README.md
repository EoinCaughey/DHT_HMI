# Arduino HMI Dashboard (Flask + Chart.js)

A real-time Human Machine Interface (HMI) dashboard for monitoring temperature and humidity data from an Arduino sensor. The system streams serial data into a Python backend and visualizes it live in a modern web dashboard using Flask and Chart.js.

---

## Features

- Live temperature and humidity monitoring from Arduino (serial input)
- Real-time line charts using Chart.js
- Web-based dashboard (runs locally in browser)
- Min / Max / Average statistics
- Timestamped sensor history
- Auto-updating UI every second
- Modern responsive UI with styled cards and layout

---

## Tech Stack

- Python 3
- Flask (backend server)
- PySerial (Arduino communication)
- Chart.js (graphing)
- HTML / CSS (dashboard UI)
- JavaScript (live updates)
